Adversarial Attack on Neural Networks
August 2017 - December 2017
Completed

Adversarial attack refer to transforming the input such that the perturbed input results in the model outputting an incorrect answer with high confidence. Experimented on Fast Gradient Method, Basic Iterative Method, Virtual Adversarial Method using the InceptionV3 model and evaluate using pre-trained using InceptionV3, InceptionV2, ResNet V2 152 over classification task.
Developed method Binary Iterative Method which performs well over most of others methods on 1000 images sampled from the standard ImageNet dataset.
